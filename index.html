Guardian Digital

Una app pensada para ayudarte a bloquear SMS spam y detectar mensajes sospechosos. Menos ruido, menos riesgos.

¿Qué hace Guardian Digital?

Ayuda a filtrar mensajes molestos o peligrosos (por ejemplo: ofertas insistentes, enlaces raros o mensajes que te meten afán). La meta es que tengas más tranquilidad al usar tu celular.

  • Bloqueo de SMS spam (publicidad repetitiva / mensajes molestos).
  • Alertas cuando un SMS tiene señales típicas de estafa.
  • Recomendaciones claras: “qué hacer” y “qué evitar”.
Quiero apoyo para instalar/configurar Preguntarle al asistente

Nota: la seguridad perfecta no existe, pero sí se puede reducir muchísimo el riesgo combinando filtros + buenos hábitos.

Consejos rápidos contra SMS spam

  • No abras enlaces si el mensaje te mete afán (“último aviso”).
  • No compartas códigos (OTP) ni contraseñas.
  • Si dice ser tu banco/entidad, confirma por el canal oficial.
  • Bloquea y reporta el número si es spam.

Si quieres, te ayudamos a dejar todo configurado paso a paso.

🤖 Chat / Asistente

Habla o escribe con un asistente

Para resolver dudas, aprender paso a paso o simplemente conversar. Incluimos “Chat con ChatGPT” en dos modos: abrir ChatGPT o chat integrado (conexión segura desde tu servidor).

💬 Chat integrado (demo)
Abrir ChatGPT
Hola 😊 Soy tu asistente. Escribe una pregunta como: “¿Cómo bloqueo mensajes spam?” o “Explícame qué es IA en sencillo”.
Nota: este chat está listo para conectarse a un endpoint seguro (sin poner claves en la página).

¿Cómo lo dejamos funcionando con ChatGPT?

Por seguridad, no se debe poner una API key dentro del HTML. Lo correcto es:

  • Tu web llama a /api/chat (en tu servidor).
  • Tu servidor llama al modelo de OpenAI con tu API key.
  • El servidor devuelve la respuesta a la página.

Si por ahora quieres algo simple, usa el botón “Abrir ChatGPT” y listo.
